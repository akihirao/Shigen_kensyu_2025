
# 水産研究・教育機構　令和７年度資源評価研修のページ

このページは水産研究・教育機構が行っている令和７年度資源管理評価研修の資料置き場のページです。 

## 本ページ概要

研修では、講義とともには統計ソフトRを使ったハンズオン演習をおこないます。ハンズオン演習で必要となるデータの説明およびRのスクリプトの一部がまとめてあります。
